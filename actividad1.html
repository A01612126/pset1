Javascript

Aquí se analizarán los tipos de variables que existen en Javascript

Existen tipos de datos primitivos y no primitivos en Javascript. Veamos qué los definen

Tipo De Dato Primitivo

Estos tipos de datos son los que ya fueron predefinidos por JS, como lo son: Number, String, Boolean, Null, Undefined, Symbol, BigInt

Number

Los "number" en Javascript son siempre guardados con un formato binario de 64 bits de doble precisión. En Javascript no se necestia declarar float, int, etc. para declarar diferentes valores numéricos

String

En Javascript, los String son similares a oraciones ya que están hechos de de una lista de caractéres.

Boolean

Representa una entidad lógica y puede tener sólo dos valores: verdadero y falso.

Null

Solo puede tener un valor y es sí mismo (null).

Undefined

Una variable a la que no se le asgina un valor se le considera como indefinida o "undefined".

Symbol

Los símbolos devuelven identificadores únicos que pueden ser usados para añadir llaves de propiedad únicas a un objeto que no haga colisión con llaves de otro código que se pueda añadir al objeto.

BigInt

Un tipo de dato que nos permite representar números enteros mayores a lo permitido con "number".

Tipo De Dato No Primitivo

Se derivan de los datos primitivos y también se les conoce como Tipos de Datos Derivados o Tipos de datos referenciados

Objetos

Son el tipo de dato más importante y conforma la base sobre la cual se construye el Javascript moderno

Bucles en Javascript

Bucle 'for'

El bucle for es uno de los bucles más comunes y versátiles en JavaScript. Se utiliza cuando se conoce exactamente cuántas veces quieres que se ejecute el código.

Bucle 'while'

El bucle while se utiliza cuando no se sabe cuántas veces se debe repetir el código, pero se repite mientras una condición sea verdadera.

Bucle 'do while'

Similar al bucle while, pero garantiza que el código se ejecute al menos una vez antes de verificar la condición.
