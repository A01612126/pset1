Introducción

Las variables son elementos fundamentales en la programación, ya que permiten almacenar y manipular datos de manera dinámica.
En esencia, una variable es un contenedor que puede contener diferentes valores durante la ejecución de un programa. La importancia de las variables radica en su capacidad para almacenar información, realizar cálculos, tomar decisiones y controlar el flujo de un programa.
Sin ellas, la programación sería estática y limitada en su capacidad para resolver problemas de manera eficiente y flexible.

Javascript

Aquí se analizarán los tipos de datos que existen en Javascript. Empecemos primero por definir los tipos de variables y sus diferencias

  • 1. var: Las variables declaradas con var tienen un ámbito de función o ámbito global, lo que significa que pueden ser accedidas desde cualquier parte de la función en la que están definidas o desde cualquier lugar del script si se definen fuera de una función.
  • 2. let: Las variables declaradas con let tienen un ámbito de bloque, lo que significa que solo están disponibles dentro del bloque en el que se declaran (por ejemplo, dentro de un bucle for, una declaración if, o un bloque de función).
  • 3. const:Las variables declaradas con const también tienen un ámbito de bloque, al igual que let. A diferencia de let, una vez que una variable se asigna utilizando const, no se puede reasignar ni redeclarar.

Ahora analizaremos los tipos de datos.

Tipo De Dato Primitivo

Existen tipos de datos primitivos y no primitivos en Javascript. Veamos qué los definen

Tipo De Dato Primitivo

Estos tipos de datos son los que ya fueron predefinidos por JS, como lo son: Number, String, Boolean, Null, Undefined, Symbol, BigInt.

Number

Los "number" en Javascript son siempre guardados con un formato binario de 64 bits de doble precisión.
En Javascript no se necestia declarar float, int, etc. para declarar diferentes valores numéricos. Ej:

var age = 30;

String

En Javascript, los String son similares a oraciones ya que están hechos de de una lista de caractéres. Ej:

var name = "John";

Boolean

Representa una entidad lógica y puede tener sólo dos valores: verdadero y falso. Ej:

var isStudent = true;

Null

Solo puede tener un valor y es sí mismo (null). Ej:

var emptyValue = null;

Undefined

Una variable a la que no se le asgina un valor se le considera como indefinida o "undefined". Ej:

var undefinedValue;

Symbol

Los símbolos devuelven identificadores únicos que pueden ser usados para añadir
llaves de propiedad únicas a un objeto que no haga colisión con llaves de otro código que se pueda añadir al objeto. Ej:

var uniqueSymbol = Symbol("description");

BigInt

Un tipo de dato que nos permite representar números enteros mayores a lo permitido con "number". Ej:

var bigNumber = 1234567890123456789012345678901234567890n;

Tipo De Dato No Primitivo

Se derivan de los datos primitivos y también se les conoce como Tipos de Datos Derivados o Tipos de datos referenciados.

Object

Son el tipo de dato más importante y conforma la base sobre la cual se construye el Javascript moderno. Ej:

var person = { name : "John", age: 30 };

Bucles en Javascript

Bucle 'for'

El bucle for es uno de los bucles más comunes y versátiles en JavaScript.
Se utiliza cuando se conoce exactamente cuántas veces quieres que se ejecute el código. Ej:

for (var i = 0; i < 5; i++) { console.log("Número: " + i); }

Bucle 'while'

El bucle while se utiliza cuando no se sabe cuántas veces se debe repetir el código, pero se repite mientras una condición sea verdadera. Ej:

var counter = 0; while (counter < 5) { console.log("Número: " + counter); counter++; }

Bucle 'do while'

Similar al bucle while, pero garantiza que el código se ejecute al menos una vez antes de verificar la condición. Ej:

var counter = 0; do { console.log("Número: " + counter); counter++; } while (counter < 5);

Conclusión

En conclusión, las variables son elementos esenciales en la programación que permiten almacenar, manipular y gestionar datos de manera eficiente. Su importancia radica en su capacidad para adaptarse y cambiar durante la ejecución de un programa, lo que proporciona flexibilidad y poder de resolución de problemas a los desarrolladores. Comprender cómo utilizar variables de manera efectiva es fundamental para convertirse en un programador competente y crear software funcional y de alta calidad.
